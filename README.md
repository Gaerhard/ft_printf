# ft_printf
Ft_printf project from 42 school, the goal of the project is to recreate the function
printf

### Installing
```
Run the make command
```

### Usage

Compile ft_printf and add libftprintf.a to yur own Makefile and ft_printf.h to your headers

### Features

* %d or %i to print integers
* %f to print floats
* %s to print strigs
* %c to print characters
* %p to print a pointer
* %b to print a binary value
* %o to print an octal value
* %u to print an unsigned int
* %x or %X to print an hexadecimal value
* #0+-* and space flags
* precision
* hh, h, l, ll and L flags
* colors
* ft_dprintf to print on a specific file descriptor
